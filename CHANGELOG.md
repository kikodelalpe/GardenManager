# Changelog

All the major updates to the project will be documented in this file.

## [1.0.1] - 2026-03-01

### Added
- **Cross-Platform Support**: Released versions for Windows, Linux and Android.
- **Database SQLite**: Implemented automatic system path detection `AppData` (Windows) and `Home` (Linux).

### Modified
- Migrated build enviornment from Linux to Windows (MinGW 13.1.0).
- Added .ico file for windows executable.

### Removed
- Removed hard path dependencies from Ubuntu filesystem.

## [1.0.0] - 2026-03-01

### Added
- **GardenManagerv1.0**: First working version of the software with botanic catalog, warehouse management and cultivation monitoring.
- **Localization**: Italian translation.
- **Data Backup**: Database backup feature.
