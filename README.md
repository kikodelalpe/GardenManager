# 🌿 GardenManager v1.0

GardenManager is a desktop application written in **C++** and **Qt 6** for the management of a greenhouse or a garden. With GardenManager you can catalog plants, manage a warehouse and monitor the cultivations.

## 🚀 Download and setup

You can download the latest version in the section [Releases](https://github.com/kikodelalpe/GardenManager/releases).

### 🪟 Windows
1. Download `Setup_GardenManager_v1.0.exe`.
2. Run setup and follow the instructions.
3. You'll find the icon on your desktop.

### 🐧 Linux
You have 2 options:
* **AppImage (Recommended):** Download `GardenManager-x86_64.AppImage`, make it executable (`chmod +x`) and run.
* **Setup script:** Download `GardenManager_Linux_v1.0.zip`, extract it and run `./install_gardenManager.sh` to add the program to the system application menu.

## 🛠️ Technologies used
* **Framework:** Qt 6.10.2
* **Language:** C++
* **Database:** SQLite (local)
* **Build System:** CMake / qmake
